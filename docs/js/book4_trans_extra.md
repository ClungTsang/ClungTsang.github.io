#  运算符优先级的概念

1. 圆括号：`( )`
2. 成员访问：`.`、`[]`
3. 函数调用/创建：`()`、`new`
4. 递增/递减：`++`、`--`
5. 一元运算符：`+`、`-`、`!`、`~`、`typeof`、`void`、`delete`
6. 算术运算符：`*`、`/`、`%`
7. 加法/减法运算符：`+`、`-`
8. 关系运算符：`<`、`>`、`<=`、`>=`、`instanceof`、`in`
9. 相等运算符：`==`、`!=`、`===`、`!==`
10. 位运算符：`<<`、`>>`、`>>>`、`&`、`|`、`^`
11. 逻辑运算符：`&&`、`||`
12. 条件运算符：`? :`
13. 赋值运算符：`=`
14. 逗号运算符：`,`
